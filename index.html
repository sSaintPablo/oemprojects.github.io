Network Vulnerability Assessment
for A More Secure Environment

This project involved a comprehensive network vulnerability assessment designed to identify, analyze, and mitigate security weaknesses within an existing IT infrastructure. Utilizing a combination of automated scanning tools (i.e Nessus, Nmap) and manual penetration testing techniques, I systematically evaluated network devices, servers, applications, and configurations for potential exploits, misconfigurations, and compliance deviations. Key activities included port scanning, service enumeration, vulnerability identification (CVE analysis), and exploit validation where permissible within a controlled environment. The outcome was a detailed report outlining identified vulnerabilities, their potential impact, and actionable remediation strategies, including patching recommendations, access control refinements, and security policy enhancements. This project directly contributed to fortifying the network's defenses, reducing its attack surface, and establishing a more robust security posture against cyber threats.
